# ieem to pep transformations

1. Espanol:

2. English:
   - `docs/ieem_to_pep_transformations_en.md`

## recommendation

- Use the Spanish version for domain discussion in Spanish.
- Use the English version for collaboration/documentation in English.
- Both versions include:
  - A symbol glossary (`I`, `J`, `K`, `L`, `AG`, `MARG`, `OTH`, `ti`, `tx`, etc.).
